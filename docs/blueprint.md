# **App Name**: Kisiangani Portfolio

## Core Features:

- Landing Page: Display an engaging landing page with an introduction and key sections.
- Service Sections: Highlight Joharie Kisiangani's services (Branding & Marketing, Website Design, Web Development, and Application Design).
- Projects Section: Showcase a selection of UX engineering projects with descriptions and links.
- Testimonials Section: Display testimonials from past clients to build credibility.
- About Section: Provide background information about Joharie Kisiangani and his approach to UX engineering.
- Contact Section: Include a contact form and relevant contact information for inquiries.
- Scroll Animation: Animate page elements on scroll for a dynamic and engaging user experience.

## Style Guidelines:

- Primary color: Dark Gray (#333333) for a professional and modern feel.
- Background color: Light Gray (#F0F0F0) to ensure content sections stand out while providing a clean backdrop.
- Accent color: Teal (#4DB6AC) to highlight important sections and calls to action, analogous to dark gray but with a higher saturation and different brightness.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and technical feel, suitable for headlines.
- Body font: 'Inter' (sans-serif) for clear and readable text in longer paragraphs. Font Pairing: 'Space Grotesk' for headlines and 'Inter' for body text.
- Minimalist icons to represent service categories and skills.
- Clean, minimal layout with clear sections for services, projects, testimonials, about, and contact.
- Subtle animations triggered on scroll to reveal content element by element.
